# viscosity_minimap

**A tablet-style bezel for the GTA minimap.** A lightweight CSS/NUI overlay that frames
the in-game minimap in a clean device bezel, styled to match a satellite/tablet map look.
Pure cosmetic — no gameplay logic.

---

## Features

- **Bezel frame** — a CSS frame drawn around the minimap.
- **Matched styling** — designed to sit nicely with a satellite minimap.
- **Zero dependencies** — standalone; just an overlay.

---

## Installation

1. Drop `viscosity_minimap` into `resources`.
2. Ensure it in `server.cfg`:
   ```cfg
   ensure viscosity_minimap
   ```

Pairs well with a high-res satellite minimap replacement.
